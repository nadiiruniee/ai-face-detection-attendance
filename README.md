# AI Face Detection

## Description
This project detects human faces in real-time using a webcam.

## Technologies
- Python
- OpenCV

## Features
- Detect face from webcam
- Draw rectangle on detected face

## Purpose
This project is part of my learning journey in Artificial Intelligence.

## How to Run
1. Install Python
2. Install OpenCV:
   pip install opencv-python
3. Run:
   python face_detection.py

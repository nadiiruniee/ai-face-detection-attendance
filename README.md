 AI Face Detection with WhatsApp Notification

📌 Overview
This project is a real-time face detection system built using Python and OpenCV. It utilizes a webcam to detect human faces and enhances functionality by integrating WhatsApp notifications when a face is detected.
This project is designed as part of an Artificial Intelligence learning journey, with practical implementation in computer vision and real-time monitoring systems.

🧠 Features
* Real-time face detection using webcam
* Face bounding box visualization
* WhatsApp notification integration
* Lightweight and fast processing
* Easy to set up and run

🛠️ Technologies Used :
* Python
* OpenCV
* WhatsApp API / Integration (Fonnte)

⚙️ Installation

* Clone this repository:
  
Bash

git clone https://github.com/nadiiruniee/ai-face-detection-attendance.git
cd ai-face-detection-attendance

* Install dependencies:
  
Bash

pip install opencv-python numpy

* (Optional) Setup WhatsApp API:
Configure your API credentials (e.g., Fonnte)
Update the script with your phone number and API key

▶️ Usage
Run the script:

Bash

python face_detection.py

* The webcam will start automatically
* Faces will be detected in real-time
* A notification will be sent via WhatsApp when a face is detected
